# 🍃 Ceylon 365

**Luxury Ceylon Tea | Pure Taste. Timeless Tradition.**

Ceylon 365 is a premium tea brand inspired by the rich heritage of Sri Lankan tea. Our mission is to deliver authentic Ceylon tea to tea lovers around the world while promoting sustainability, quality, and innovation.

---

## 🌍 About the Brand

Ceylon 365 represents the finest hand-selected tea leaves grown in the lush highlands of Sri Lanka. Every cup reflects generations of craftsmanship, ethical sourcing, and exceptional quality.

### Brand Values

- Premium Quality
- Authentic Ceylon Tea
- Sustainable Sourcing
- Innovation
- Customer Satisfaction

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/ceylon365.git
```

### Navigate to the project

```bash
cd ceylon365
```

### Open locally

Simply open `index.html` in your preferred browser.

Or use VS Code Live Server:

```bash
Right Click → Open with Live Server
```
